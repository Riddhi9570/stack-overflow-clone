# stack-overflow-clone
- This a repository of [Stack Overflow](https://stackoverflow.com/) clone, made for learning purpose.
- Languages and libraries used : HTML, CSS, Javascript, React.js

# Usage
## Client
- Use `cd client` to change the working directory to client.
- Use `npm install` to install dependencies.
- Use `npm start`to run the client.

## Server
- Use `cd server` to change the working directory to server.
- Use `npm install` to install dependencies.
- Use `npm start` to run the server.
